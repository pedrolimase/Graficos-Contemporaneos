Diagrama de Arcos

O que um Diagrama de Arco ?

Um diagrama de arco um tipo especial de grfico de rede. uma representao grfica unidimensional para visualizar conexes entre um conjunto de entidades. A ideia principal exibr os ns ao longo de um eixo, enquanto as conexes entre os ns so representadas com arcos.

Grficos interessantes

Les Miserables

Gaston criou o diagrama de arco abaixo para visualizar os personagens do clssico de Victor Hugo, que se tornou um filme, Les Miserables. Cada personagem conectado por um arco se eles aparecerem juntos no mesmo captulo; quanto maior o arco, mais os personagens apareceram nos captulos juntos. A ordem (e a cor) dos ns identificam grupos de caracteres que aparecem juntos no romance.

Mapa de conversas do Twitter

Esta visualizao foi criada por Walter Rafelsberger,essa visualizao mostra as conversas de cerca de 1500 usurios do servio de microblog Twitter. Os arcos basicamente ligam os locais dos usurios que conversam entre si. A geocodificao foi feita filtrando as informaes de localizao das pginas de perfil dos usurios e pesquisando-as com nomes geogrficos.

Uso bsico do pacote Arcdiagram

A estrutura simples e consiste em uma funo principal arcplot(), e uma funo acessria xynodes(). Arcplot funciona da seguinte maneira, pega um objeto edgelist e plota as arestas como arcos. Edgelist apenas uma matriz de duas colunas que fornece uma lista de arestas para um grfico. Arcplot funciona da seguinte maneira, pega um objeto edgelist e plota as arestas como arcos.

library(arcdiagram)
# Cria um grfico com 8 ns
star_graph = graph.star(8, mode = "out")

#Adiciona nome aos ns
V(star_graph)$name = paste("node", 1:vcount(star_graph), sep = "-")

# Extrair edgelist
star_edges = get.edgelist(star_graph)

# plot arc diagram
arcplot(star_edges)

Criando um grfico

Se quisermos plotar nosso grfico usando o pacote igraph(), primeiro precisamos criar um objeto graph com a funo graph.edgelist() e, em seguida, podemos usar o mtodo plot() padro:

# Grfico da edgelist
glab = graph.edgelist(lab, directed = TRUE)
# plot grfico
plot(glab)

Podemos levar em considerao o grau dos ns - o grau de um n o nmero de arestas conectadas, para isso usaremos degrees():

lab_degree = degree(glab)
## Emilia   Kirk   Yong Filipe Matteo  Tyler Mehmet   Rori  Vitor   Anna 
##      2      2      2      3      3      2      2      3      1      2

Digamos que queremos colocar pesos nas arestas para que elas reflitam algum tipo de valor. Ns podemos fazer isso atribuindo alguns nmeros aleatrios:

arcplot(lab, lwd.arcs = l)

Tambm podemos obter clusters dos ns usando a funo clusters():

gclus = clusters(glab)

Aplicando cores que reflitam as associaes de cluster:

# Definindo dois tons de azul
blues = c("#adccff", "#4272bf")
# Vetor de cores com base na associao dos clusters
cols = blues[gclus$membership]

Aplicando a funo arcplot()

arcplot(lab, lwd.arcs = l, cex.nodes = lab_degree,
col.nodes = cols)

lwd.arcs: largura das linhas dos arcos

cex.nodes: tamanho dos smbolos dos ns

Vamos ordenar os ns de acordo com os clusters:

arcplot(lab, lwd.arcs = l, col.nodes = cols, 
        cex.nodes = lab_degree, bg.nodes = cols, ordering = order(gclus$membership))

col.nodes: cor dos smbolos dos ns

bg.nodes: cor de preenchimento para os smbolos dos ns

ordering: vetor numrico opcional que fornece a ordem dos ns

Vamos mudar o diagrama para uma orientao vertical:

arcplot(lab, lwd.arcs = l,
cex.nodes = lab_degree, col.nodes = cols, bg.nodes = cols,
show.nodes = TRUE, ordering = order(gclus$membership), horizontal = FALSE)

show.nodes: indica se deseja mostrar smbolo nos ns

horizontal: indica se deseja traar na orientao horizontal

Desvantagens

Os arcos so sensveis ordem dos ns. Quando os ns relacionados esto prximos uns dos outros, tem-se uma melhor visualizao, o que ajuda a detectar clusters ou grupo de ns.
